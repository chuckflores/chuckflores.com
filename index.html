Sketch App

Sketch is one of my favorite design tools for creating wireframes and digital design assets. With InVision and Framer integration I can quick produce prototypes and iterate upon the designs within Sketch.

Tips / Tricks

  • When setting up a project for integating within Invision declare a set naming structure for your artboards. I use a number system like so: 00-00-00-00. Each number can represent a specific flow, page, interaction, or different screen states. This is helpful when it comes time to build the prototype. You will be able to build the flow very quickly by looking at the progression of numbers 01-01-01-01 → 01-01-01-02 → 01-01-01-03 etc…
